Скрипт парсит коммутаторы серии (Catalyst, SG300) в открытом контуре (internet) на наличие портов со статусом "admin down" ,"shutdown".

Как пользоваться:
1. Установить Python не ниже версии 3.8
2. Установить модули. В командной строке:
     pip install regex
     pip install netmiko 
4. В командной строке указать путь к папке со скриптом и запустить:
    C:\MyScript\>python main.py
5. Следовать дальнейшим инструкциям.
